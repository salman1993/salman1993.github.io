## site / blog

## Installing locally:
- Clone the repo
- Run: `bundle install`

## Running locally:
- [Docs](https://docs.github.com/en/pages/setting-up-a-github-pages-site-with-jekyll/testing-your-github-pages-site-locally-with-jekyll)
- Run: `bundle exec jekyll serve --verbose`
- Navigate to `http://localhost:4000`
